# Lema Markdown

Fork personal de [MarkText](https://github.com/marktext/marktext), un editor de Markdown open-source, elegante y enfocado en velocidad y usabilidad.

Este proyecto parte del código original de MarkText (licencia MIT) y lo mantiene como base propia para experimentación y desarrollo independiente.
